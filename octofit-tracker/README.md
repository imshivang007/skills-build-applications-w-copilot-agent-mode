# Octofit Tracker

OctoFit Tracker is a multi-tier fitness app for student activity logging, team challenges, and progress tracking.

## Project structure

- `backend/` for the API and logic tier
- `frontend/` for the presentation tier
- `docs/octofitapp-small.png` for the app logo

## Stack

- React 19 with Vite
- Express with TypeScript
- MongoDB with Mongoose

## Next steps

- Install dependencies in `backend/` and `frontend/`
- Start the API on port `8000`
- Start the UI on port `5173`
